# 3D_print_3.25_speaker_enclosure
File STL thiết kế thùng loa 3.25" dùng để in 3D

Link youtube
- https://youtu.be/k8h7kqgMxKA
- https://youtu.be/5omHe2NPuSA

Mình in với chế độ:
- độ dày thành: 1.2mm
- top/bot: 1.2mm
- độ dày lớp: 0.3mm
- vòi: 0.4mm
- infill: 35%

Tổng khối lượng vào khoảng 600g, thời gian in khoảng 24h
